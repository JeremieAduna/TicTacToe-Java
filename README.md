# TicTacToe-Java

A console-based implementation of the classic Tic Tac Toe game using Java.  
Two players take turns placing Xs and Os on a 3Ã—3 board until one wins or the game ends in a draw.

## Features

- 2-player turn-based gameplay
- Win and draw detection
- Simple console interface
- Clear code structure using core Java logic
